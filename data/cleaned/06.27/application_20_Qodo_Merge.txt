Qodo Merge

**簡介**

一種協助開發者更快、更高效地審查 Pull Request (PR) 的AI代理工具。它會自動分析提交和 PR，並提供：
 -自動描述：自動生成 PR 描述，包括名稱、類型、摘要和代碼演示。
 -PR 審查：提供關於 PR 主題、類型、相關測試、安全問題、重點 PR 以及各種建議的反饋。
 -問題解答：回答關於 PR 的自由文本問題。
 -代碼建議：提供可提交的程式碼建議以改進 PR。
